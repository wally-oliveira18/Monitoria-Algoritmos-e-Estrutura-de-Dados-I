# Monitoria Algoritmos e Estrutura de Dados I

Este repositório reúne os códigos desenvolvidos durante minha atuação como monitor da disciplina Algoritmos e Estrutura de Dados I, ministrada no curso de Sistemas de Informação da PUC-MG. Os exemplos foram implementados em C# e abordam os principais conteúdos da disciplina, incluindo fundamentos de programção, como estruturas condicionais, estruturas de repetição, vetores, matrizes, struct, classes, métodos, encapsulamento, construtores.
